Win7Keys - Enable Windows 7 shortcuts on XP and Vista

Download Browse Source

Summary

This application enables Windows 7 shortcuts on Windows XP and Windows Vista. It's a tiny no-nonsense app that just works.

Keys Implemented

  1. Win + Left/Right - Dock active application to the left/right side
  2. Win + Up - Maximize active application
  3. Win + Down - Minimize active application
